# Face_Mask_Detection

This code consists of various libraries which is used for image classifications. And the main aim of the code is to detect the mask
on an human face and also tells the percentage of the perfection of detection, and if there is no mask detected then enclose the face 
with a red color boundry and specify that there is no mask.

A program in python made by using OpenCV, Keras and Tensorflow, which is used to detect the person's face whether he/she is with mask or without mask.
